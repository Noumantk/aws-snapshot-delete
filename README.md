# AWS-Snapshot-Delete
A Python script used to delete AWS snapshots matching the supplied criteria

## Usage Examples
`python3 delete_snapshots.py --delete --verbose --age 7` - delete all snapshots that are older than 7 days

## Options
* --filter , -f :
* --profile , -p :
* --age , -a :
* --delete , -d :
* --verbose , -v :
* --quiet , -q :

## Requirements
* boto3
